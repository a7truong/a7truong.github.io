    WLP4 Compiler

    Compiler Design.

    This compiler handles WLP4 programs; the WLP4 language is a subset of the C++ programming language. WLP4 consists of pointers, pointer arithmetic, integer, arithmetic, and procedures. This compiler was built to recognize the language through as series of analysis with syntactic and sematic errors before code generation. Syntactic analysis consisted of parsing and rule analysis. The sematic analysis included type checking and any other error. The final stage of the compiler was to generate code recognized by a MIPS machine. A high-leveled program was transformed into basic MIPS assembly language to compute the task.

    • Date: April 2015
    • Category: Compiler design.

    PowerShell

    Scripting.

    Windows Powershell is a terminal environment running on Microsoft Window machines. Powershell is a powerful tool to run commands and scripts to automate and manage users and computers across a network. Some scripts created were implemented into production stages for automatic processes on online servers. These servers host Active Directory, which manage users, groups, and computers.

    WinSCP Automated File Transfer | The WinSCP file transfer script was created to automate the file transfer process between a Windows Server to a Linux Server. The script would check for a file on the Windows box (specific path) and would notify users via e-mail if the file was present or not. If the file was present and the WinSCP file transfer was successful, an e-mail would be sent out indicating a successful transfer from Windows to Linux.

    • Date: July 2015

    User Principal Name Change | The user principal name (UPN) change script was created to run on AD servers to automate the changes the UPN to the user's respective first and last name. As implemented before, the UPN was the user's employee number; as the migration to Microsoft Office 365, UPNs were required to be in the form "first.last" as the prefix. The suffix of the UPN was the network domain.

    • Date: June 2015

    Active Sync Device Detection | This script was created to indicate the entire network communicating devices. The script would connect to the Exchange Management Console and gather user data along with the devices associated with their account. Once the script is complete, an email is sent to an administrator indicating all users with at least two communicating devices on their profile.

    • Date: June 2015

    Password Notification | The Password Notification script was developed in order to help users across the Region of Peel be aware of their soon-to-expire password. This script grabs each Active Directory (AD) user on the network and checks their account between the last set password date to the current date. If the difference is progressing to at most 7 days, there will be daily emails being sent out. The emails being sent out are connected to the mailhub server, and are sent starting from the seventh day counting down to the day the password must be changed.

    • Date: October 2014
