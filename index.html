    WLP4 Compiler

    Compiler Design.

    This compiler handles WLP4 programs; the WLP4 language is a subset of the C++ programming language. WLP4 consists of pointers, pointer arithmetic, integer, arithmetic, and procedures. This compiler was built to recognize the language through as series of analysis with syntactic and sematic errors before code generation. Syntatic analysis consisted of parsing and rule analysis. The sematic analysis included type checking and any other error. The final stage of the compiler was to generate code recognized by a MIPS machine. A high leveled program was transformed into basic MIPS assembly language to compute the task.

    • Date: April 2015
    • Category: Compiler design.
